🆕 New: IPFS Image Storage
Every photo uploaded is stored on IPFS — a decentralized file network. The image's unique fingerprint (CID) is recorded on the blockchain. This means the photo can never be swapped or tampered with. Consumers see the actual photo of their food's origin.

⚡ Setup Guide
  1. Run: npx hardhat node
  2. Deploy: npx hardhat run scripts/deploy.js --network localhost
  3. Update CONTRACT_ADDRESS in app.js
  4. Get free IPFS API key from nft.storage or pinata.cloud
  5. Update API key in ipfs.js
  6. Open in browser with MetaMask on Hardhat localhost (Chain ID: 31337)
